Custom directives¶

Directives for page¶

These directives are only one per pages.

.. pyscript-env::¶

Define PyScript runtime environment as py-env.

Example:

.. pyscript-env::

   - beautifulsoup4
   - requests

Directives for content¶

.. pyscript-repl::¶

Render PyScript REPL as py-repl.

Example:

.. pyscript-repl::

   print("hello world")
